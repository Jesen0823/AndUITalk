
如何快速精确搜索github开源项目

in:name 条件: 根据名字查找
in:name django
in:readme 条件 : 根据项目说明过滤
in:readme django
in:description 条件: 根据项目描述过滤
in:description django
language:条件: 根据语言过滤
in:description django language:python
pushed: 条件 ： 根据推送时间过率
in:description django language:python puished:>2020-04-03
stars: 条件: 根据点赞数来过滤
in:description django stars:>1000
